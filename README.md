# \# Phone to QR Code Generator

# 

# A simple, browser-based tool to convert any phone number into a clickable QR code. Perfect for printing contact information or sharing numbers quickly.

# 

# \## Features

# \* Generate QR codes instantly for any phone number.

# \* Supports international format (+ country code).

# \* Save the QR code as a PNG or JPG.

# \* Includes a built-in print feature.

# 

# \## How to use

# 1\. Open the \[index.html](index.html) file in your browser.

# 2\. Enter the phone number (e.g., +971 50 123 4567).

# 3\. Click "Generate QR Code".Phone2QRCode\_Generator
